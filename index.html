Introduction to Computer Hardware

IT Workshop: A Deep Dive into Computer Components

An Introductory Guide for 1st Year B.Tech Students

What is a Computer?

  • An electronic device that processes data.
  • It takes in raw data (input), processes it, and provides a result (output).
  • This is all made possible by a combination of Hardware and Software.
  • Hardware: The physical components you can touch.
  • Software: The set of instructions that tells the hardware what to do.

The Core Components

Today, we'll look at the essential hardware components:

  • Motherboard
  • Central Processing Unit (CPU)
  • Memory (RAM & ROM)
  • Storage (HDD, SSD, NVMe)
  • Power Supply Unit (PSU/SMPS)
  • Input & Output Devices

The Motherboard - The Backbone

  • The main circuit board of the computer.
  • Everything connects to the motherboard.
  • It allows all components to communicate.
  • Think of it as the central nervous system.

The CPU - The Brain

  • The "brain" of the computer.
  • Performs calculations and executes instructions.
  • Speed is measured in Gigahertz (GHz).
  • Modern CPUs have multiple "cores" for multitasking.

Memory - RAM & ROM

RAM (Random Access Memory)

  • Volatile: data is lost when power is off.
  • Used for short-term data storage for running apps.

ROM (Read-Only Memory)

  • Non-volatile: data persists without power.
  • Contains firmware like BIOS/UEFI.

Storage - Your Digital Library

HDD (Hard Disk Drive)

Uses spinning magnetic disks. Larger capacities, lower cost.

SSD (Solid State Drive)

Uses flash memory. No moving parts. Much faster.

NVMe SSD

The newest and fastest type of SSD. Connects directly to the motherboard.

PSU / SMPS - The Heart

  • SMPS: Switched-Mode Power Supply.
  • Converts AC wall power to DC for components.
  • Provides necessary voltage and current.
  • The "heart" that pumps "blood" (power).

Input & Output Devices

Input Devices

How you give instructions to the computer.

  • Keyboard, Mouse, Webcam, etc.

Output Devices

How the computer gives you information.

  • Monitor, Printer, Speakers, etc.

How It All Works Together

  1. You provide Input.
  2. The CPU processes this input.
  3. The CPU uses RAM for temporary storage.
  4. Data is moved to/from Storage.
  5. The result is shown via an Output device.
  6. All powered by the PSU & connected by the Motherboard.

Summary

  • Motherboard: The main connector.
  • CPU: The brain that processes everything.
  • RAM: Short-term memory for active tasks.
  • Storage: Long-term memory for your files.
  • PSU: Provides power to all components.
  • Input/Output Devices: Your interface.
